## 📖 项目简介 (Introduction)

这是一个旨在自动评估数字绘画线条质量的自动化评分系统。项目主要基于线条的**平滑度 (Smoothness)** 和**连续性 (Continuity)** 算法来对用户的绘画线条进行打分。

This is an automated scoring system designed to evaluate the quality of digital drawing lines. The project primarily assesses lines based on algorithms for **smoothness** and **continuity**.
